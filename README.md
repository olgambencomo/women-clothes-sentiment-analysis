#  Women Clothes Reviews — Sentiment Analysis

This project focuses on analyzing customer reviews from a women’s clothing e-commerce store to uncover insights about customer satisfaction, sentiment trends, and product perception.  
The analysis was conducted using **Python (Jupyter Notebook)** for Natural Language Processing (NLP) and **Power BI** for interactive data visualization.

---

## Project Overview

Customer reviews contain valuable feedback that reflects how buyers perceive product quality, sizing, and service.  
This project applies sentiment analysis to classify reviews as **positive, neutral, or negative**, and then visualizes key trends through a Power BI dashboard.

---

## Tools & Technologies

- **Python**: pandas, nltk, TextBlob / VaderSentiment/ RobertaBase 
- **Power BI**: Data modeling, DAX calculations, and interactive dashboards  
- **Jupyter Notebook** for data cleaning and NLP processing
